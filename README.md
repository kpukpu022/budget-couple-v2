# Budget Couple 💸

App de budget collaboratif pour deux personnes.

## Install

```bash
npm install
npm run dev
```

## Deploy sur Vercel

1. Push ce repo sur GitHub
2. Va sur vercel.com → "Add New Project"
3. Importe le repo GitHub
4. Clique Deploy — c'est tout !

## Installer sur iPhone

1. Ouvre l'URL Vercel dans **Safari**
2. Bouton Partager ↑ → "Sur l'écran d'accueil"
3. Guillaume fait pareil avec la même URL
